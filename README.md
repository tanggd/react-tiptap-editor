# react-tiptap-editor
React component based on Tiptap


创建项目。
基于tiptap、react19、vite、ts、antd、less封装tiptap编辑器，导出2个组件，一个编辑器组件，一个视图组件。
增加代码规范，增加eslint9.x、husky、prettier、stylelint、lint-staged的组合能力。
编辑器的工具栏基于antd组件实现。编辑器的工具栏有：
- 撤销
- 重做
- 加粗
- 斜体
- 下划线
- 删除线
- 标题（h1-h6）
- 列表（有序、无序、任务）
- 引用
- 代码块
- 行内代码
- 图片
- 链接
- 颜色（字体颜色、背景颜色）
- 上标
- 下标
- 居左
- 居中
- 居右
- 插入表格
- 插入公式
- 清除格式
